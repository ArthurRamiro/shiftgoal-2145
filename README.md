# ShiftGoal 2145

Bem-vindo a ShiftGoal 2145, um jogo de futebol arcade 2D com estética cyberpunk, gravidade dinâmica e partidas intensas para dois jogadores locais.

## 🎮 Sobre o jogo

ShiftGoal 2145 mistura ação rápida, física invertida e estratégia em tempo real. O objetivo é simples: marcar gols, controlar a gravidade e vencer o adversário em uma arena futurista.

### Destaques
- Multiplayer local para 2 jogadores
- Gravidade dinâmica que muda o rumo da partida
- Estilo visual cyberpunk e neo-futurista
- Modo de jogo rápido e viciante
- Classificação indicativa: Livre para todos os públicos

## 🕹️ Como jogar

### Jogar online
- Versão no GitHub Pages: https://arthurramiro.github.io/shiftgoal-2145/
- Link alternativo no itch.io: https://arthur-ramiro.itch.io/shiftgoal-2145

### Jogar localmente
1. Baixe o arquivo ShiftGoal_2145_JOGAR.html
2. Abra o arquivo em qualquer navegador moderno
3. Clique em “Iniciar Partida” e aproveite o jogo

## 🎮 Controles

### Jogador 1 - JAX
- A: mover para a esquerda
- D: mover para a direita
- W: pular / impulsionar

### Jogador 2 - VAILE
- ←: mover para a esquerda
- →: mover para a direita
- ↑: pular / impulsionar

### Comandos adicionais
- ESC ou P: pausar a partida

## ⚙️ Requisitos

- Navegador moderno com suporte a HTML5
- Sem necessidade de instalação adicional para a versão local
